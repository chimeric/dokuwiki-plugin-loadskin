====== Template Manager ======

This page allows you to specify different templates for different pages/namespaces in your wiki.

The page check is recursive, that means, if you specify a different template for a wiki page ''wiki'' all other pages inside the ''wiki:'' namespace will use this template as well.

**Important note:** In order to make the plugin work correctly you have to follow the instructions at [[http://wiki.splitbrain.org/plugin:loadskin#installation]].
